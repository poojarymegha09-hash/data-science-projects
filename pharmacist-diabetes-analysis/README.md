# Pharmacist-led Monitoring in Type 2 Diabetes

This project analyzes data from a study evaluating the impact of pharmacist-led monitoring on patients with Type 2 Diabetes.

The objective of the analysis is to explore clinical outcomes and assess how pharmacist interventions may improve patient monitoring and treatment outcomes.

## Tools Used

- R
- Statistical analysis
- Data visualization

## Files

- AnalysStat.R – statistical analysis of patient data
- visualisation.R – visualizations of key variables and outcomes

## Analysis Performed

- Descriptive statistics
- Clinical outcome comparison
- Data visualization of patient variables
