# Hospital Infection Data Analysis and Machine Learning

This project analyzes a hospital-acquired infections dataset using Python. The analysis includes exploratory data analysis (EDA) and machine learning models to explore patterns in healthcare data and predict potential infection outcomes.

## Objectives
The goal of this project is to understand patterns in clinical variables such as age, hospital stay duration and infection indicators, and to apply machine learning models for predictive analysis.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Analysis Performed
- Data loading and cleaning
- Missing value analysis
- Descriptive statistics
- Exploratory data analysis
- Data visualization
- Machine learning modelling
- Model evaluation
